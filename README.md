# デバッグログ

プログラミング学習の失敗と不安を共有するサービス

## サービス概要
デバッグログは、プログラミング学習における「失敗・後悔・迷い」を共有できるサービスです。  
卒業生や学習経験者の実体験から、これから学ぶ人がより良い選択をできるようにします。

## このサービスを作ろうと思った理由
プログラミングスクールで約9ヶ月学習する中で、AIの使い方、学習方法、転職活動の進め方などで多くの迷いや後悔を経験しました。  
こうした失敗は自分だけでなく、多くの学習者が経験する一方で、体系的に共有される場が少ないと感じました。  
そこで、失敗や不安を次の人の判断材料に変えられるサービスを作りたいと考えました。

## 想定ユーザー
- プログラミングスクール受講生
- 独学で学習している人
- 学習方法や転職活動に不安がある人

## 提供したい価値
- 他人の失敗から学べる
- 学習や転職の判断材料が増える
- 同じ後悔を防げる

## MVP機能
- ユーザー登録 / ログイン
- 投稿機能
- 投稿一覧表示
- 投稿詳細表示
- 匿名投稿機能

## 使用技術
- Ruby on Rails
- PostgreSQL
- TailwindCSS
- Render

## 技術選定理由
### PostgreSQL
SQLiteは学習初期には扱いやすい一方で、ローカル開発向きの側面が強いと考えました。  
MySQLも有力な選択肢ですが、本アプリではRailsとの相性や今後の拡張性を重視しました。  
PostgreSQLは、ユーザーと投稿の関連を安全に管理しやすく、検索や絞り込みなどの機能追加にも対応しやすいため選定しました。
